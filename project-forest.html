Forest Canopy Prediction

Python Machine Learning TensorFlow

Project Overview

This project leverages the power of machine learning and satellite imagery to address critical environmental challenges. By analyzing vast datasets of satellite photos, the model can accurately identify and predict changes in forest canopy density, helping to monitor deforestation, track reforestation efforts, and assess overall forest health. This tool provides valuable insights for environmental agencies and conservation groups.

Key Features

  • Automated processing of high-resolution satellite imagery.
  • A deep learning model (CNN) trained to classify forest density.
  • Data visualization dashboard to display trends and predictions over time.
  • API for integrating the prediction model with other environmental monitoring tools.

My Contribution

In this project, I was responsible for the core machine learning pipeline. My work involved sourcing and pre-processing the satellite image data, which included cleaning, augmenting, and labeling thousands of images. I then designed, trained, and fine-tuned the convolutional neural network (CNN) using TensorFlow to achieve high accuracy in canopy prediction. I also developed scripts for data analysis and contributed to the backend logic for the visualization dashboard.
