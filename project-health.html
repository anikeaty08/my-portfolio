Maternal Health Web App

JavaScript Figma Hackathon

Project Overview

This web application was developed during a hackathon with the mission to improve maternal healthcare access in tier 2 and tier 3 cities. The platform provides expectant mothers with a simple, intuitive tool for regular health check-ups, pregnancy tracking, and access to vital information. The project was recognized for its social impact and user-centric design, ultimately winning the hackathon.

Key Features

  • User-friendly interface for daily health logging (e.g., symptoms, vitals).
  • Personalized dashboard to track pregnancy milestones and progress.
  • Educational resources and articles on maternal wellness.
  • Appointment reminders and scheduling with local healthcare providers.
  • Emergency contact feature for immediate assistance.

My Contribution

During this fast-paced hackathon, I took the lead on UI/UX design and frontend development. I began by creating high-fidelity mockups and a complete design system in Figma to ensure a cohesive and intuitive user experience. I then translated these designs into a responsive and interactive web application using vanilla JavaScript, HTML, and CSS. My focus was on creating a clean, accessible interface that could be easily used by individuals with varying levels of digital literacy.
